vue-init使用私有模板

发表于 2018-11-28 | |

通常做法

在使用vue-cli快速创建项目时，普遍做法是fork一份vuejs-templates/webpack然后再做修改。

私有git仓库

通常做法对于github上没有私有仓库但又有保密需要的人来说，不太友好。寻找了一番后，发现vue-init支持其他的git托管服务: https://github.com/vuejs/vue-cli/issues/525#issuecomment-344504097。
具体下来就是：vue-init -c <host>:<userName>/<repo> <projectName>

# vue-init # private # template
