vue-init使用私有模板

发表于 2018-11-28 |

通常做法

在使用vue-cli快速创建项目时，普遍做法是fork一份vuejs-templates/webpack然后再做修改。

私有git仓库

通常做法对于github上没有私有仓库但又有保密需要的人来说，不太友好。寻找了一番后，发现vue-init支持其他的git托管服务: https://github.com/vuejs/vue-cli/issues/525#issuecomment-344504097。
具体下来就是：vue-init -c <host>:<userName>/<repo> <projectName>

关键点解析

  1. NR, FNR

    • awk官方说明：

      NR: The total number of input records seen so far
      FNR: The input record number in the current input file

    • 当NR==FNR时，是在读取feed.txt文件，并将关键列的值塞入array中；

    • 当NR>FNR时，在读取full_data.txt文件，碰到关键列值存在时输出整行print $0

  2. getValue函数

    • column ~ /^[0-9]+$/

      判断是否为单纯的列

    • column ~ /^[0-9]+_[0-9]+$/

      判断是否为可并列，按照_分割得出index，再把相应的值拼装起来
